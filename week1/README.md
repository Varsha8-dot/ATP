(if / else)
Used to make decisions in code.
The program checks a condition and runs a block of code if it's true, otherwise runs the else block.
Useful for comparing values and controlling the flow of a program.

Arrays & Array Methods
An array is a collection of values stored in a single variable. JavaScript provides built-in methods to work with arrays:

splice() – Used to add, remove, or replace elements at a specific position in an array.
find() – Searches through an array and returns the first element that matches a given condition

An object stores related data as key-value pairs (e.g., name, age, marks).
Arrays of objects let you manage multiple records together, like a list of employees with their details.


(for loop)
A loop repeats a block of code a set number of times.
The for loop is commonly used to go through each element in an array — for example, to calculate a sum or find a specific value
