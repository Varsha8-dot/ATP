# HTML & CSS Basics Project

This project demonstrates how HTML and CSS are connected to build a simple web page.

## What I Learned

* Structure of a webpage using HTML
* Styling using CSS
* Different ways to apply CSS:

  * Inline CSS
  * Internal CSS
  * External CSS

## Files Included

* `index.html` → Main HTML file
* `style.css` → External CSS file

## How It Works

The HTML file is linked with the CSS file using the `<link>` tag inside the `<head>` section.
CSS is used to change colors, layout, and design of the webpage.

## Output

A simple styled webpage with background color and text styling.
